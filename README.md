# Configuration

## Install PHP

https://www.php.net/

## Edit service configuration

1) click edit configuration at top right
2) click '+' icon on top left
3) select PHP build-in web service
4) set Host to 'localhost'
5) set port to '8000'
6) set Document root to project root
7) click ok
